## Name
Image Measurer

## Description
Laden und Bearbeiten von verschiedenen Bildateien und das Vermessen von verschiedenen Objekten des Bildes. Beim Vermessen werden Metrische Eintheiten verwendet mindestens mm, cm, m und km.
